inicio Etiquetas más usadas Estilos

Estructura básica

Todas las páginas de nuestra web (y de la mayoría) deben empezar y terminar por la etiqueta <html> y terminar con </html> (aunque además, se puede poner antes una etiqueta de comentario <!DOCTYPE html> para facilitar la correcta interpretación de todos los navegadores).

Entre ellas se colocan las etiquetas de las dos partes de una página web: la cabeza y el cuerpo: <head> y <body>, cada una con su correspondiente etiqueta final.

La parte de la cabeza contiene las partes de la página que no deben ser visibles, y se cargan antes que la parte visible, el cuerpo. Contiene, por ejemplo, archivos que se cargan antes, como los de estilo, y trozos de script -programas que se ejecutan dentro de la página web.

El cuerpo contiene el texto, espacios, botones e imágenes que se pueden ver.

En resumen:

    <html>
         <head>
            <!-- Aquí va lo que no se ve.-->
         </head>
         <body>
            <!-- Aquí va la parte que se puede ver.-->
         </body>
    <html>

Cargando una hoja de estilos css

Un archivo css es una lista de palabras que definen estilos de formato de clases de etiqueta.

Aunque el lenguaje html nos permite poner el formato dentro de una etiqueta web, esto hace que sea más engorroso crear la página web, porque:

  • Vemos peor el contenido importante.
  • Los estilos de un párrafo pueden ser distintos de otro, por error al escribir el mismo código muchas veces en cada etiqueta.
  • Si queremos cambiar el estilo de la web, hay que hacer cambios en muchas etiquetas en vez de en una, y esto supone tiempo y errores.

La etiqueta <link> permite cargar un archivo con definiciones de estilo. Dentro hay que poner el nombre del archivo en el parámetro href y el tipo de archivo en el parámetro rel.

<link href="mi_archivo.css" rel="stylesheet">

Hay que sustituir mi_archivo.css por el nombre correcto de nuestro archivo, que, además, deberíamos colocar dentro de una carpeta css para que tengamos bien organizados los archivos de nuestra página web.
