Etiquetas más usadas

Cabeceras o encabezados

Toda web debe tener encabezados claros, como cualquier documento. Los encabezados se hacen con la etiqueta <hX>, en la que la X es un número que marca el nivel de importancia. <h1> es el de primer nivel. <h6> es el más pequeño. Hay que poner una etiqueta donde empieza, y la misma con diagonal / cuando termina.

Los encabezados inician una línea nueva y terminan con una línea nueva.

Divisiones y párrafos

Párrafos

El párrafo es un trozo de texto que puede contener varias frases, y que empieza en una nueva línea.

No hace falta indicar este salto final. Las etiquetas <p> y </p> se encargan de colcar este salto.

Si fuera necesario podemos separar forzar un salto obligatorio dentro de un párrafo mediante la etiqueta <br>. Aunque se genera una nueva línea, ambas pertenecen al mismo párrafo.

División

La división se coloca entre las etiquetas <div> y </div>. Genera un bloque parecido al párrafo, pero, en este caso, se suele usar para meter dentro otros tipos de contenidos. Es una especie de contenedor.

Lo más habitual es aplicarle estilos, para así poder formatear la página con más facilidad.

Ejemplo:

        <div style="padding-left: 40px; padding-right: 40px; color: red">
    Este párrafo está formateado como una división, a la que se 
    ha dado el estilo de espacio a los lados de 40 puntos
    y color rojo para el texto.
</div>
Este párrafo está formateado como una división, a la que se ha dado el estilo de espacio a los lados de 40 puntos y color rojo para el texto.

En general debemos aplicar los estilos dentro del archivo css, tal y como veremos en la página que explica este tema. Pero en este ejemplo se ha colocado el estilo dentro de la etiqueta, a modo de parámetro.

Spans

Un span es un bloque de texto que está dentro de una línea. Sirve para enmarcar las palabras o caracteres que contiene de una manera especial:

El agua del mar se dice que tiene color azul, pero es a veces un tono entre cyan y aguamarina, y otras veces gris azulado oscuro.

Enlaces

Para hacer que un texto se convierta en un hiper-enlace hay que usar la etiqueta <a>, dentro de la cual pondremos el parámetro href para escribir la dirección de destino:

<a href="cuatro.html">Pincha aquí</a>

Imágenes

La etiqueta <img>insertará una imagen en la página web.

Esta etiqueta exige que pongamos la ruta al archivo de imagen en el parámetro src.

<img src="ies_solo_logo.png">

También podemos utilizar los parámetros width y height para indicar el tamaño de la imagen, aunque otra opción es usar una definición dentro del archivo css de estilos, si el tamaño de todas las imágenes es el mismo.

Otros elementos html

También son muy usadas las etiquetas table, code, blockquote y script. Sirven, respectivamente:

  • para hacer tablas (se explicó en la primera página),
  • para formatear el texto como si fuera un ejemplo de programación (está utilizado en todas las páginas para poner ejemplos),
  • para crear un párrafo que parezca una cita,
  • y, el último, para escribir un programa que se ejecute con la página web, como por ejemplo los fragmentos javascript que generan botones y acciones con ellos.

Las enumeraciones y listas se hacen con las etiquetas <ul> -si es con puntos como la anterior-, y <ol> -si están numeradas-. Por supuesto con su correspondiente etiqueta para terminar la lista.

Cada elemento de estas listas se debe escribir dentro del conjunto de etiquetas <li> y </li>.
